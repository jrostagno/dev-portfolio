## Compact Rules

### ui-ux-pro-max

**When**: Any component creation, styling decision, layout review, or accessibility check.

**Standards**:
- Design source of truth: `DESIGN.md` — all tokens, colors, spacing, components defined there
- Dark-only palette: bg `#0A1528`, surface `#122648`, primary `#3B82F6`, text `#CDD7EA`
- Typography: Inter only, weights 300/400/500/600/700, `font-display: swap`
- Radius: 12px (inputs/chips), 16px (cards), 999px (pills/buttons)
- Spacing: 4/8px base rhythm; section gaps 48–56px
- Motion: 150–300ms, ease-out, transform+opacity only; respect `prefers-reduced-motion`
- Accessibility: WCAG 2.2 AA, focus rings 2px solid `--primary-2`, min touch target 44×44px
- Mobile-first, no horizontal scroll, safe-area insets on iOS
- Lucide icons only — no emoji as structural icons
- Use CSS custom properties (no hex values in components)
- CTAs: one primary action visible per screen
